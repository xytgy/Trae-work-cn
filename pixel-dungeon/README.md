# 像素地牢：枪战冒险

一款像素风格的地牢射击游戏，玩家在随机生成的地牢中闯关，体验紧张刺激的射击战斗，最终击败Boss通关。

## 🎮 操作说明

| 操作 | 按键 |
|------|------|
| 移动 | WASD / 方向键 |
| 射击 | 鼠标左键（自动瞄准最近敌人） |
| 切换武器 | Q |
| 暂停 | P / ESC |

## 🔫 游戏特色

- **多种武器**：手枪、闪电法杖、榴弹发射器、火焰喷射器、星星回旋镖
- **多样化敌人**：绿色史莱姆、红色蝙蝠、紫色幽灵
- **独特Boss战**：三阶段Boss，50%和25%血量触发阶段变化
- **武器掉落**：击杀敌人30%概率掉落武器
- **像素风格**：致敬经典的像素美术

## 🚀 运行方式

直接在浏览器中打开 `index.html` 即可开始游戏。

## 🎯 游戏流程

1. **开始游戏** → 进入房间1，有2个史莱姆
2. **闯关** → 击杀所有敌人 → 获得武器掉落 → 进入下一房间
3. **Boss战** → 房间4后Boss登场 → 击败Boss通关
4. **重玩** → 死亡或通关后可重新开始
