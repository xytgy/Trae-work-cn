# 《像素地牢：枪战冒险》游戏设计规格说明书

**版本：** 1.0
**日期：** 2026-06-30
**项目类型：** 像素风格地牢射击游戏
**技术栈：** HTML5 Canvas + 原生JavaScript

---

## 一、游戏概述

### 1.1 一句话描述
一款像素风格的地牢射击游戏，玩家在随机生成的地牢中闯关，体验紧张刺激的射击战斗，最终击败Boss通关。

### 1.2 核心玩法循环
```
进入房间 → 击杀所有敌人 → 获得武器/道具 → 进入下一个房间 → Boss战 → 通关
```

### 1.3 游戏特色
- 🎯 自动瞄准射击，简化操作
- 🔫 多种武器随机掉落
- 👾 多样化敌人 + 独特Boss
- 💎 像素风格，致敬经典

## 二、玩家系统

### 2.1 操控方式
| 操作 | 按键 |
|------|------|
| 移动 | WASD / 方向键 |
| 射击 | 鼠标左键（自动瞄准最近敌人） |
| 切换武器 | Q键 |

### 2.2 玩家属性
| 属性 | 数值 |
|------|------|
| 生命值 | 3颗心 |
| 伤害 | 受攻击时失去1颗心 |
| 无敌帧 | 受伤后1秒无敌时间（角色闪烁） |
| 初始武器 | 手枪（无限弹药） |
| 最大武器携带 | 2把 |

### 2.3 玩家状态
| 状态 | 说明 |
|------|------|
| 正常 | 可正常操作 |
| 无敌 | 受伤后1秒，角色闪烁，不可被攻击 |
| 死亡 | 生命值为0，显示死亡画面 |

## 三、武器系统

### 3.1 武器列表

| ID | 名称 | 伤害 | 射速 | 弹药 | 特点 | 图标 |
|----|------|------|------|------|------|------|
| 1 | 手枪 | 1 | 中 | 无限 | 初始武器，稳定输出 | 🔫 |
| 2 | 闪电法杖 | 2 | 快 | 30发 | 发射电弧，可穿透1个敌人 | ⚡ |
| 3 | 榴弹发射器 | 3 | 慢 | 15发 | 发射炮弹，有爆炸范围伤害 | 💣 |
| 4 | 火焰喷射器 | 1 | 极快 | 50发 | 短距离火焰，持续伤害 | 🔥 |
| 5 | 星星回旋镖 | 2 | 中 | 20发 | 发射后返回，可攻击多个敌人 | ⭐ |

### 3.2 武器拾取规则
- 击杀敌人后概率掉落武器（掉落率30%）
- 拾取新武器时：
  - 如果当前武器栏未满，直接拾取
  - 如果已满，显示切换提示，选择保留哪把
- 武器弹药耗尽后自动切换回手枪

### 3.3 武器切换
- 按Q键在两把武器间切换
- 切换有0.3秒冷却时间
- 当前武器显示在UI底部

## 四、敌人系统

### 4.1 普通敌人类型

#### 4.1.1 绿色史莱姆（小怪A）
| 属性 | 数值 |
|------|------|
| 血量 | 2 |
| 移动速度 | 慢（1.5像素/帧） |
| 伤害 | 1（接触） |
| 掉落率 | 30% |

**AI行为：**
- 持续向玩家移动
- 与玩家距离小于30像素时进行接触攻击
- 攻击冷却时间1秒

#### 4.1.2 红色蝙蝠（小怪B）
| 属性 | 数值 |
|------|------|
| 血量 | 1 |
| 移动速度 | 快（3像素/帧） |
| 伤害 | 1（接触） |
| 掉落率 | 25% |

**AI行为：**
- 快速向玩家移动
- 接近玩家后快速后退，反复横跳
- 攻击冷却时间0.5秒

#### 4.1.3 紫色幽灵（小怪C）
| 属性 | 数值 |
|------|------|
| 血量 | 2 |
| 移动速度 | 中（2像素/帧） |
| 伤害 | 1（接触或投射物） |
| 掉落率 | 35% |

**AI行为：**
- 缓慢向玩家移动
- 每隔2秒发射一个投射物
- 投射物速度：4像素/帧

### 4.2 敌人通用行为
- 自动追踪玩家（路径：直接向玩家移动）
- 与玩家保持最小攻击距离（30像素）
- 被击中时播放受伤动画（闪红）
- 死亡时播放消失动画（缩小+透明）

## 五、Boss系统

### 5.1 Boss外观
- 体积：96x96像素（普通敌人3倍大小）
- 外观：大型紫色地牢守护者
- 像素风格设计

### 5.2 Boss属性
| 属性 | 数值 |
|------|------|
| 血量 | 20 |
| 移动速度 | 1像素/帧 |
| 碰撞伤害 | 2 |

### 5.3 Boss攻击阶段

#### 阶段1：正常状态（HP 20-10）
| 攻击方式 | 描述 | 冷却时间 | 视觉效果 |
|----------|------|----------|----------|
| 散弹攻击 | 发射5发子弹向四周散开 | 3秒 | 红色子弹 |

**阶段1行为：**
- 缓慢追踪玩家
- 停止时发射散弹
- 子弹伤害：1

#### 阶段2：狂暴状态（HP 10-5，HP≤50%触发）
| 变化 | 描述 |
|------|------|
| 外观变化 | 颜色变红，身体发光 |
| 召唤小怪 | 每5秒召唤2个小怪A |
| 移动速度 | 提升至1.5像素/帧 |

#### 阶段3：最终状态（HP 5-0，HP≤25%触发）
| 变化 | 描述 |
|------|------|
| 外观变化 | 全身发光更强 |
| 激光扫射 | 每8秒执行一次激光攻击 |
| 激光躲避 | 激光持续2秒，需要走位躲避 |

**激光攻击详情：**
- 激光宽度：40像素
- 激光颜色：黄色
- 伤害：持续伤害，每0.2秒1点
- 预警：激光出现前有0.5秒红色警告线

### 5.4 Boss死亡
- 播放爆炸粒子特效
- 显示击杀得分
- 掉落大量武器和道具
- 进入通关结算画面

## 六、关卡设计

### 6.1 关卡结构
```
房间1 → 房间2 → 房间3 → 房间4 → Boss房间 → 通关
```

### 6.2 房间规格
| 属性 | 数值 |
|------|------|
| 游戏区域尺寸 | 800 x 600像素 |
| 墙壁厚度 | 40像素 |
| 可活动区域 | 720 x 520像素 |
| 门位置 | 中央（所有方向） |

### 6.3 各房间敌人配置

| 房间 | 敌人配置 | 特殊设置 |
|------|----------|----------|
| 房间1 | 2个小怪A | 新手引导房间 |
| 房间2 | 2个小怪A + 1个小怪B | 混合战斗 |
| 房间3 | 1个小怪A + 2个小怪B + 1个小怪C | 远程敌人加入 |
| 房间4 | 2个小怪A + 2个小怪B + 2个小怪C | 压力测试 |
| Boss房间 | Boss + 召唤的小怪 | 最终战 |

### 6.4 房间转换
- 击杀房间内所有敌人后
- 房间中央出现传送门（3秒后可用）
- 玩家进入传送门进入下一房间
- Boss房间无传送门，击杀Boss即为通关

```

├── index.html          # 游戏入口页面
├── css/
│   └── style.css       # 样式文件
├── js/
```

- 模块化设计，便于维护
